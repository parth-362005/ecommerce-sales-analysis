## Problem Statement
To analyze e-commerce sales performance and customer behavior using data visualization techniques in Power BI, in order to identify key trends, patterns, and insights that can support business decision-making.

## Dataset Description
The dataset contains information about order date, sales, category, sub-category, region, and customer details.

## Tools Used
- Power BI
- Python
- CSV Dataset

## Project Files
- Dashboard
- Report
- Presentation
- Dataset

## Key Insights
- Technology category generated the highest sales
- West region showed strong sales performance
- Monthly trends revealed variation in demand

## Conclusion
This project helped in understanding e-commerce business performance through visual analysis and insights.
